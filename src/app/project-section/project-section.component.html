GitHub Issue Browser (2021)

What was initially a take-home coding assessment ended up being a personal project of mine. It had asked me to create a web application that would allow users to look up issues relating to a GitHub repository.

I believed that the assessment provided a real-world problem and that a solution could be useful for real users, so I continued working on it after the deadline had passed. It was originally built using Python's Django web framework and utilized GitHub's API to fetch data.

I had recently re-built the project using Angular 11 and TypeScript with Angular Bootstrap components. The UI is cleaner, simpler, and provides a seamless experience for users.

Budgeter (2020)

Ever since I was about 12 years old, I had a strong interest in personal finance. I would ask my dad to be a personal bank and see if he could give interest on whatever I had given him to hold.

The interest rate he had given me for holding on to my money was around 10% and let me tell you, that rate spoiled me when I got older and realized that interest rates for savings accounts were no where near that.

My interest in personal finance and has grown over the years and I wanted to take a shot at creating my own web application that allows users to track their spending.

This was the first web application that I had ever built and it was great fun! I learned a lot about HTML, CSS, SQLite3, and the Django framework. I was able to implement a lot of features that I would have never imagined when I first started the project.

Food Recipe Web Scraper (2020)

I absolutely love cooking and it's one of my favorite hobbies. There's just something special about being able to create something that brings people together. Perhaps that's why I enjoy being a software developer!

This project was developed in Python and used Regular Expressions to scrape the web to find recipes based on the input ingredients. I decided to implement this program into my Discord Bot that I had been working on. I had a lot of fun working on this project because it was solving a problem that I could relate to!

Often times, I have a certain set of ingredients available at home and would like to conjure up a dish based on those ingredients. Working on this project really put into perspective how programs, software, and technology have the ability to solve common, relatable, everyday problems.
